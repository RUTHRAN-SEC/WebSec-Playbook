# SQL injection

## Description
SQL Injection (SQLi) is a critical web security vulnerability that occurs when an attacker manipulates database queries through unsanitized user input. This can allow unauthorized access to sensitive information such as usernames, passwords, credit card details, and personal data. In severe cases, attackers may modify or delete database content, bypass authentication, gain administrative access, or even compromise the underlying server infrastructure. SQL injection attacks can lead to data breaches, financial loss, reputational damage, and long-term system compromise.
