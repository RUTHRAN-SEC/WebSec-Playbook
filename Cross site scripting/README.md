# Cross-site scripting

## Description
Cross-Site Scripting (XSS) is a web security vulnerability that allows attackers to inject malicious JavaScript into a trusted website. When a victim accesses the vulnerable application, the malicious script executes in their browser, enabling the attacker to steal session cookies, access sensitive data, impersonate users, or perform actions on behalf of the victim. XSS bypasses the browser’s same-origin policy and can lead to account compromise and full application control if the targeted user has elevated privileges.
