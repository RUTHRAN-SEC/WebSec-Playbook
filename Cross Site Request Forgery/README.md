# Cros Site Request Forgery

## Description
CSRF is a web security vulnerability that tricks an authenticated user into unknowingly performing unintended actions on a trusted website. 
It works by exploiting the fact that a browser automatically includes session cookies or credentials in requests, allowing an attacker to forge requests that appear legitimate.

In a CSRF attack, the victim is lured into visiting a malicious page that silently triggers actions such as changing account details, 
updating passwords, or making financial transactions on a logged-in application. If the user has privileged access, 
the attacker may gain control over the account or even the entire application.
